# Property_Registry 🏡

## Project Description
The Stellar Property Registry is a decentralized application built on the Stellar network using the Soroban smart contract platform. It provides a secure, immutable, and transparent way to register, manage, and transfer real estate or digital property ownership entirely on-chain.

## What it does
This smart contract allows users to mint a digital representation of a property by assigning it a unique ID, a physical address, an initial monetary value, and a designated owner. Once registered, the contract enforces cryptographic authorization, ensuring that only the rightful owner can transfer the property to a new Stellar address. Anyone on the network can query the contract to instantly verify the current owner and the details of a specific property.

## Features
- **Register Property:** Securely create a new property record bound to a specific Stellar address.
- **Transfer Ownership:** Cryptographically transfer the property from one owner to another (requires the current owner's digital signature to execute).
- **Query Data:** Fetch the current state of a property, including its physical address, registered value, and current owner.
- **Double-Registration Prevention:** Built-in validation checks to ensure a property ID cannot be registered twice, preserving data integrity.
